## 插件简介

Server酱评论通知插件是一款适用于 Typecho 的插件。当博客有新评论时，插件会通过 Server酱推送通知到您的微信或其他支持的渠道，帮助您及时了解博客动态。

## 功能特点

- **实时通知**：在有新评论时立即推送通知。
- **高度可定制**：支持自定义通知标题、内容和标签。
- **多种格式支持**：兼容 Server酱 SC3 和 SCT 两种 SendKey 格式。
- **忽略博主评论**：可选择忽略博主自己的评论通知。

## 安装与配置

### 安装步骤

1. 下载插件文件。
2. 将插件文件上传至 Typecho 的 `usr/plugins` 目录。
3. 登录 Typecho 后台，进入“控制台 > 插件管理”。
4. 找到“Server酱评论通知插件”，点击“启用”。
5. 进入插件设置页面，填写 Server酱的 SendKey 和其他配置项。

### 配置说明

- **SendKey**：Server酱的密钥，可从 [SendKey 页面](https://sc.ftqq.com/) 获取。
- **通知标题**：支持使用占位符（如 `{author}`、`{postTitle}` 等）动态生成标题。
- **通知内容**：支持 Markdown 格式，可自定义推送消息的正文。
- **标签（Tags）**：为推送消息添加标签，便于分类管理。
- **忽略自己的评论**：启用后，博主自己的评论不会触发通知。

## 使用方法

启用插件并完成配置后，插件会在有新评论时自动推送通知，无需额外操作。

## 常见问题

### 1. 如何获取 SendKey？
请访问 [Server酱官网](https://sc.ftqq.com/)，登录后生成您的 SendKey。

### 2. 为什么没有收到通知？
- 请检查 SendKey 是否填写正确。
- 确保服务器能够访问 Server酱的推送接口。
- 检查插件设置是否正确，尤其是“忽略自己的评论”选项。
